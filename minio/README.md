# MinIO

High-performance, S3-compatible object storage for any workload.

## What's Included

- **MinIO Bucket** — S3-compatible object storage

## Use Cases

- **Application assets** — Store user uploads, images, and files
- **Backups** — Automated backup storage for databases and services
- **Data lake** — Store large datasets for analytics and ML
- **Static hosting** — Serve static files directly from storage

## Connecting

Other workloads can access the bucket using the injected S3-compatible credentials. Works with any S3 client library.
