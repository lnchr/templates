# Uptime Kuma

A self-hosted monitoring tool with beautiful status pages and flexible alerting.

## What's Included

- **Uptime Kuma** — Monitoring application (port 3001)
- **Volume** — Persistent storage for monitoring data

## Getting Started

After deploying, visit your domain to create an admin account. Then start adding monitors for your services.

### Supported Monitor Types

- HTTP(s) / Keyword / JSON query
- TCP / Ping / DNS
- Docker container status
- Push / Heartbeat
- Game server (Steam, Minecraft)

### Alerting

Supports 90+ notification services including Slack, Discord, Telegram, Email, PagerDuty, and more.
