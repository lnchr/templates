# WordPress

The world's most popular content management system, powering over 40% of websites.

## What's Included

- **WordPress** — Latest WordPress with Apache, PHP, and wp-cli
- **MySQL** — Database backend
- **Volume** — Persistent storage for wp-content (themes, plugins, uploads)

## Getting Started

After deploying, visit your assigned domain to run the WordPress installer. You'll set up your site title, admin account, and language.

### Customizing

- Install themes and plugins via the admin panel at `/wp-admin`
- Upload media files — they persist across restarts via the attached volume
- Configure caching, security, and SEO plugins as needed
