# Ghost

A professional publishing platform with built-in memberships, newsletters, and content monetization.

## What's Included

- **Ghost** — Publishing app running on Alpine Linux (port 2368)
- **MySQL** — Database backend for content and member data

## Getting Started

After deploying, visit your domain to complete the Ghost setup wizard. You'll create an admin account and configure your publication name.

### Features

- Rich editor for posts and pages
- Built-in membership and subscription system
- Newsletter support with email delivery
- Custom themes and integrations
- SEO and social sharing tools
