# Gitea

A lightweight, self-hosted Git service — the painless way to set up your own code hosting.

## What's Included

- **Gitea** — Git web interface with CI/CD (port 3000)
- **PostgreSQL** — Database for users, repos, and metadata

## Getting Started

After deploying, visit your domain to complete the Gitea installer. The database settings are pre-configured.

### Features

- Git repository hosting with web UI
- Pull requests, issues, and project boards
- Built-in CI/CD with Gitea Actions (compatible with GitHub Actions)
- Package registry for Docker, npm, Maven, and more
- Webhooks and API for integrations
